== Description ==

This plugin created for you will be able to create an online shop on your wordpress site.
You can add "buy" button for different post. Excellent ecommerce.

[youtube https://www.youtube.com/watch?v=9VvSZ2da6h4]

== Installation ==

1. Upload `yocommerce` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add paypal api creadentials to plugin's settings
